My plan with this site is to document my progress by updating it with new things I learn, and adding a snapshot of the current state at the end of every chapter.

Chapter Two: Adding CSS

25.11.2021

In this chapter I am focusing on CSS and I'm going to do my best to make the site readable and easy to digest.

CSS can be applied inline (not recommended), internal (inside the html file, also not recommended), or external (for example styles.css, this is what I will focus on)

Things to note while learning CSS:

  • using selectors
  • combining selectors to apply rules
  • learning about the specificity of selectors
  • learning about inheritance

The built-in inspect tool of my browser was a huge help while figuring out what the different css commands did (especially margin, padding, border). Using it on sites I like inspired me to make something similar.
